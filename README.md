# Daily Streak

A simple daily streak tracker built with vanilla JavaScript.  
Progress is saved locally in the browser using `localStorage`.

## Features
- Increase streak count for a successful day
- Reset streak on failure
- Data persists after page refresh or browser restart
- No login, no backend

## Tech Stack
- HTML
- CSS
- JavaScript
- Browser localStorage

## How to Use
1. Open the app in your browser
2. Click **Success** if you complete the day
3. Click **Reset** if you fail the day
4. Your streak is saved automatically

## Notes
This project is meant for personal habit tracking and learning purposes.
